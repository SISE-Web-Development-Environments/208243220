Ans

1. 3 differences between web1, web2 and web3:
 1. Dynamic of the web:
In web1: the pages are only static.
In web2: pages are more dynamic, where a person can exchange information with others.
In web3: the pages are dynamic like web2 but can also do refresh more often than web2

2. The purpose of the web:
In web1: the purpose was to connect information. Companies were the only ones who controlled the data in the web with very little interaction between consumers.
In web2: the purpose was to connect people. The users could participate in the web, they were able to search and share their data with all other users with less control.
In web3: the purpose is to connect knowledge. The focus is on making data open and accessible to all. The Web contains a variety of data sets for the customers to use and users can interact with others.

3. The way search engines works:
In web1: search engines retrieved macro contents. The search was fast but in most cases the results were inaccurate.
In web2: search engines retrieved micro contents according to tags. The process of tagging was manual and covered only little amount of the web.
In web3: search engines will retrieve micro content texts which were tagged automatically. This will increase significant the amount of retrieval documents.
It can also help to make more precise search. because tagging can solve part of the ambiguity that exist in the queries.

2. Medicines side effects Use Case:
In web1: The user enter the query "side affects of medicines". Then the search engine search and displays to the user only the pages that the query expression appear in the name.
In web2: The user enter the query "side affects of medicines". Then the search engine search and displays to the user only the pages that the query expression appear in the content of the page.
In web3: The user enter the query "side affects of medicines". Then the search engine search and displays to the user all of the pages that contains content that related to the query expression.
